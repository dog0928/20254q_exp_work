# プロダクト開発: 10年後のZEN Study

## コンセプト
- AIを組み入れられてより学習の効率と質が向上したオンライン学習プラットフォーム
- 現状の問題点が解決され、UXが大幅に改善されたサービス
- VR/AR技術を活用した没入型学習体験

## 担当別ワーク

### 概要・ターゲット・ビジネスモデル
-  概要
 ZEN Studyは、幅広い分野の学習とN高グループのレポートを動画・ライブ授業・フォーラムを通じて自分のペースで学べるオンライン学習プラットフォームである。
- 学習履歴や学力診断による可視化により、学習の理解度と成長を継続的に支援する。 
- 10年後のZEN Studyは、AI・VR・ARなどの技術を活用して、誰もが自分に合った学び方を選べる総合学習プラットフォームへと進化している。
- 学習履歴や学力診断、動画による振り返り、レポート評価の即時化により、学習成果がすぐに可視化・フィードバックされる点が中核的な価値である。

-  ターゲット
① 中学生・高校生（メインターゲット）
- 中学内容の復習から大学受験レベルまで、学力差を問わず学べる層
- 通学コース・ネットコース双方の生徒
- 部活動・課外活動と学習を両立したい生徒
特に、「自分のペースで学びたい」
「一度で理解できず、動画で何度も見返したい」
「学習時間や進捗を可視化して管理したい」
といったニーズを持つ生徒にとって、ZEN Studyは「学習の基盤」となっている。
- ② 自律的に学び直したい若年層・社会人
- 大学受験の学び直し
- 教養・分野横断的な学習
- VR教材による疑似体験型学習

-  ビジネスモデル
 10年後のZEN Studyは、教育×テクノロジー×データを軸とした複合的なビジネスモデルを構築している。
① サブスクリプション型学習プラットフォーム
- 教科学習、ライブ授業、アーカイブ教材への定額アクセス
- 中学復習〜大学受験までを一気通貫で提供
- ダークテーマ対応・字幕機能など、アクセシビリティを重視したUI
② AIによる学習支援・自動化
- フォーラムの質問回答はAIが一次対応
- 学力診断・学習履歴から最適な教材・学習時間を提示
- 学習に必要な時間の自動表示によるスケジュール最適化
これにより、教員・運営側の負担を軽減しつつ、学習の質と即時性を高めている。
③ VR・AR教材を活用した付加価値提供
- VRによる体験型教材（実験・フィールドワーク・部活動アーカイブなど）
- ARグラス対応による没入型学習体験
- 通学コースの授業通知・連動機能
これらをオプションまたは上位プランとして提供することで、単なる学習サービスではなく「未来型学習体験」としての差別化を実現している。

### 画面の構成・UI
- MVVMモデルを採用し、各画面の役割を明確化する
- NLobyと統一感のあるUIにすることで、ユーザーが直感的に操作できるようにする
- ダークテーマとホワイトテーマの切り替えが可能
- UI表示→情報を表示することで、待ち時間を最小限に抑える
- モバイルアプリでは、ネイティブのメニューを活用し、OSに最適化された操作性を提供する
- VR/ARモードで、現実世界を見つつ学習コンテンツにアクセス可能
- 学習に必要な時間を表示する
- 動画に字幕が表示されるようにする
- 通学コースの授業スケジュールを確認できる
- レポートのスケジュール管理が可能
- 長文のレポートの評価が即時行われてフィードバックをもらえる

### 機能一覧
- フォーラムのAIオートレスポンス

    質問投稿後、AIが回答のヒントを提示。解決しない場合は人間が回答。
- 没入型「タイムトラベル授業」

    歴史や科学の重要場面をVRで体験。体験中の行動ログがそのままテスト評価に反映。
- 即時フィードバック型レポート提出

    記述式レポートを提出した瞬間に、AIが「論理構成の弱点」を指摘し、再提出までのガイドを行う。
- 学習体験の拡張

    視線検知型インスタント・アシスト 教材内の不明な単語を「注視」するだけで、AIがフォーラムの過去ログや解説動画を即座にポップアップ表示。
- UIの最適化

    デバイスフリー・シームレスUI PC・モバイル間の操作感の差を排除し、学習環境を問わず直感的に利用できる統一的な画面構成。
- 学習コンテンツの拡張

    全世代対応型リカレント・コンテンツ 実務に直結する専門知識やリベラルアーツなど、社会人の学び直しにも対応した高度な教材ライブラリの拡充。

- アプティブな学習
    中学復習から大学受験まで、学力診断結果に基づき、RPG形式で進行するパーソナライズ教材。理解度に応じてストーリーや難易度がリアルタイムに変化し、最短ルートで目標に到達させる。
- 生成AIによる個別最適化した教材の作成
    - 生徒ごとに難易度・例え・分量が異なる教材を自動生成
    - 「教科書が合わない」問題を解消
- 理解の可視化ダッシュボード
    - 学習量・理解度・集中度・成長曲線を一目で確認
    - 数字だけでなく「できるようになったこと」を言語化
